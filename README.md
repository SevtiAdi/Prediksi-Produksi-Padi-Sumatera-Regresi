# Prediksi Produksi Padi Sumatera (Regresi)

## 📌 Deskripsi

Proyek ini bertujuan untuk memprediksi produksi tanaman padi di wilayah Pulau Sumatera menggunakan metode Machine Learning berbasis regresi. Model yang dibangun diharapkan dapat membantu dalam analisis dan perencanaan produksi pertanian.

---

## 🎯 Tujuan

* Menganalisis data produksi padi di Sumatera
* Membangun model prediksi menggunakan regresi
* Mengevaluasi performa model
* Memberikan insight terhadap faktor yang mempengaruhi produksi padi

---

## 📊 Dataset

Dataset yang digunakan:

* `Data_Tanaman_Padi_Sumatera_version_1.csv`

Dataset ini berisi informasi terkait produksi padi di beberapa wilayah di Sumatera.

---

## ⚙️ Metode

Metode yang digunakan dalam proyek ini:

* Regresi (Machine Learning)
* Data preprocessing
* Visualisasi data
* Evaluasi model

---

## 🛠️ Tools & Library
